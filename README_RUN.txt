========================================
  推荐运行方式
========================================

方式一：使用新的简化脚本（推荐）
---------------------------------
直接双击运行：

  run_PID_simple.bat         -> 运行 PID 模式
  run_Lagrangian_simple.bat  -> 运行拉格朗日模式
  run_RL_simple.bat          -> 运行 RL 模式

这些脚本：
✓ 自动配置参数
✓ 清理旧数据
✓ 编译项目
✓ 启动所有节点
✓ 无中文乱码问题


方式二：手动配置 + 原始脚本
---------------------------------
1. 编辑 paramsConfig.json
2. 修改 JustitiaSubsidyMode:
   - 5 = PID 控制器
   - 6 = 拉格朗日优化
   - 7 = 强化学习
3. 双击 complie_run_IpAddr=127_0_0_1.bat


方式三：命令行运行
---------------------------------
# 配置模式
copy paramsConfig_PID.json paramsConfig.json

# 编译
go build

# 运行（参考 complie_run_IpAddr=127_0_0_1.bat 中的命令）


========================================

  快速测试流程
========================================

2. 等待所有窗口启动
3. 等待实验完成（窗口会自动关闭）
4. 查看 expTest_PID/result/ 中的结果

========================================
  常见问题
========================================

   重新运行脚本即可

Q: 如何停止实验？

Q: 如何对比不同模式？
A: 依次运行三个脚本，对比三个结果目录
